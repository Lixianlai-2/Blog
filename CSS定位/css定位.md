## 两个问题

### 背景的范围是从哪里到哪里？

![背景的范围是从哪里到哪里](背景的范围是从哪里到哪里.png)

- 是从 border 外边延围城的区域
- 背景颜色时包括 border 的，不过被 border 的设置给覆盖掉了，也就是外边缘围成的区域
- 先选中 border 的颜色，然后按下 alt，往下的方向键数字减少，往上的方向键数字增加,就能
